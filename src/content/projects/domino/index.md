# Data centric
데이터 중심의 작업 방식을 지원합니다. 이를 달성하기 위해선 Rig를 구성하는 각 부분을 개별 Data 로 In, Out 할 수 있어야 합니다. domino 는 Guide 뿐만 아니라 pre-post scripts, skin, blendShape, deformerWeights, spaceManager, PSDManager, SDKManager 등으로 다양한 데이터를 In, Out 할 수 있습니다. 

# Modular Rigging System
각각의 Module 로 다양한 구조를 만들 수 있습니다.

# Guidable Rigging System
Rig는 Guide와 직접적으로 연결되어 있습니다. Guide를 움직이면 변경 사항이 Rig에 바로 반영됩니다. 수정이 필요 없는 Guide는 언제든지 Detach 할 수 있습니다.
